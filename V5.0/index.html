Robot Edits

            

          

          

            
            
          

          

            

              
              

                
Robot Export sends selected LS files from the open project to the robot selected in the top Online control. A robot with the FANUC ASCII Program Loader option will compile each uploaded LS file into a TP program.

                

                  
Required Robot Options

                  
    
                    
  • FTP Interface (R796) - transfers LS files and verifies robot files.
    • 
                    
  • ASCII Program Loader (J716) - compiles uploaded LS files into TP programs.
    • 
                  

                  
ASCII Upload is not required.

                  

                    
FTP Interface (R796)Not checked

                    
ASCII Program Loader (J716)Not checked

                  

                

                
Important: Exporting can replace programs already stored on the robot. Programs with unsaved Robo Programmer edits are exported using the current editor contents. Never export while the target program is running or selected for execution.

              

            

            

              

                
                
                
                
                
              

              
Go Online, then check the online robot before selecting programs to export.

Back Up Files From A Robot

              
Robot Backup reads selected files through the robot's FTP Interface (R796) and saves them as a timestamped ZIP inside this project's Robot Backups folder.

              
    
                
  1. Choose a backup type, then inventory the online robot.
    2. 
                
  2. Press Inventory Robot Files. Every matching file is selected by default.
    3. 
                
  3. Review the list, then use individual checkboxes, Select All, or Clear Selection.
    4. 
                
  4. Press Back Up Selected Files and wait for the ZIP to be written into the project.
    5. 
              

              
Important: All Available Robot Files includes files exposed through FTP. It is not a controller image backup and may not be identical to the teach pendant's All of Above backup.

            

            

              

                
                
                
                
              

              
Go Online, then inventory the robot before selecting files to back up.
